# keygen

### Prerequisites

- nodejs
- npm

### Local setup

- `npm i`
- configure the desired private key passphrase in the `keygen.js` line 4

```js
const certPassphrase = "ALPHA BRAVO CHARLIE DELTA ECHO FOXTROT";
```

### Run

- `npm start`

The following files should be written as output:

- public.pem
- private.pem
- passphrase
